THE ROUNDTABLE - Claude Skill
==============================
Your AI panel of real estate's best minds, inside your own Claude.

INSTALL (Claude Code / Claude Desktop with skills):
1. Unzip this file.
2. Copy the "roundtable-coach" folder into your skills directory:
     Windows:  C:/Users/<you>/.claude/skills/
     Mac:      ~/.claude/skills/
3. Restart Claude Code. Ask anything real estate - or type /roundtable-coach.

INSTALL (claude.ai web - Pro/Max/Team with Skills enabled):
1. Unzip this file.
2. In claude.ai: Settings -> Capabilities -> Skills -> Upload skill.
3. Upload the "roundtable-coach" folder (zipped) there.
